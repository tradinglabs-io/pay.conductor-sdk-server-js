# OpenapiClient::PostOrdersRequestSession

## Properties

| Name | Type | Description | Notes |
| ---- | ---- | ----------- | ----- |
| **expires_in_minutes** | **Float** | Time in minutes for checkout session expiration | [optional][default to 30] |
| **checkout_url** | **String** | Checkout URL for redirection | [optional] |
| **session_id** | **String** | ID generated by our service to identify the checkout session | [optional] |
| **external_session_id** | **String** | Checkout session ID generated by the client side. The ID must be unique for each cart session | [optional] |

```ruby
require 'openapi_client'

instance = OpenapiClient::PostOrdersRequestSession.new(
  expires_in_minutes: null,
  checkout_url: null,
  session_id: null,
  external_session_id: null
)
```
